统信命令

  • 查看系统版本

    • 命令查询：
      cat /etc/os-version（使用终端输入）

    • 图形界面查询：
      设置–》控制中心–》系统信息

  • 查看商店源命令

    • 终端输入：
      cat /etc/apt/sources.list.d/appstore.list

  • 查看系统激活状态

    • 终端输入：
      uos-activator-cmd -q
      ​

  • 激活系统命令：

    • uos-activator-cmd -a XXXXX序列号
      使用序列号命令行激活统信，其余使用uos-activator-cmd –help

  • 下载安装软件安装包

    • 使用终端下载

      • apt search 【软件名】
        使用这个命令搜索想对应的安装包名称

      • apt-get download 【包名】
        ​使用这个命令可以将软件的安装包下载下来，一般默认的下载路径是主目录

      • apt –help
        查看是其他命令的使用方法，可以在线查看

  • 安装软件

    • 使用应用商店安装软件
      打开应用商店搜索软件名称，点击下载

    • 使用终端在线安装

      • apt-get install 【包名】
        可以在线安装软件

      • apt-get reinstall 【包名】
        可以重新安装软件，在线安装

    • 使用终端离线安装

      • sudo dpkg -i 【包名】
        使用dpkg命令安装软件包，sudo是给他权限，如果提示没有权限，使用Open-root,或者使用 debug-shell 安装

  • 清理软件缓存

    • 可执行如下两条命令清理缓存：
      rm -rf ~/.local/share/Kingsoft
      rm -rf ~/.config/Kingsoft
      这里的～后面都为软件的缓存路径

  • 卸载组件

    • apt purge [组件名称]
      卸载组件
